Project Description

This project presents a 1990s video game recommendation system styled after the early web aesthetic of the same era. Designed to capture the spirit of the time, the system provides personalized game recommendations based on user preferences across multiple categories, offering a dynamic and fun way to revisit the golden age of gaming.

Using a dataset of 150 popular video games from the 1990s, the project leverages a JSON-based database to match user preferences in categories like genre, publisher, release platform, and more. The interactive interface employs sliders to refine recommendations, allowing users to discover games that align with their interests. When insufficient data is available for a specific preference, the system provides thoughtful fallback messages, ensuring transparency about its limitations.

The project wasn’t without its challenges. Building the dataset required iterative refinement, as ChatGPT struggled with filtering games strictly by decade, occasionally including post-1999 titles. Manual intervention and tailored prompts resolved these discrepancies, highlighting the subjective nature of classification. Debugging the HTML and JavaScript for the interface was another hurdle, as ChatGPT intermittently truncated the code output, requiring incremental troubleshooting. Despite these frustrations, the process underscored the complexities and limitations of generative AI as a tool for collaborative creation.

Inspired by Safiya Noble's discussion of subjectivity in classification (Algorithms of Oppression), the project reflects on how genre choices and data structuring influence recommendations. For instance, genres were curated with user experience in mind, adding broader categories like “Action/Adventure” to accommodate games that defy traditional classification. This decision-making process mirrors the subjective deployment of categories in algorithmic systems, demonstrating how even seemingly neutral tools are shaped by human intention and cultural context.

Ultimately, 90s Video Game Rater blends nostalgia with functionality, serving as both a playful tool for retro gaming enthusiasts and a thoughtful reflection on how recommendations are shaped by design choices.
